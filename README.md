## ğŸ”§ Troubleshooting

### Common Issues

#### 1. Database Connection Error

```bash
# Check PostgreSQL is running
pg_isready -h localhost -p 5432

# Verify DATABASE_URL format
# postgresql://username:password@localhost:5432/database_name
```

#### 2. CORS Errors

Ensure `CORS_ORIGIN` in backend `.env` matches your frontend URL:
```env
CORS_ORIGIN=http://localhost:3000
```

#### 3. Gemini Rate Limiting (429 Errors)

```env
# Use free tier settings
GEMINI_TIER=free
# Or upgrade your API tier
GEMINI_TIER=paid
```

#### 4. Virtual Environment Activation (Windows)

```bash
# Git Bash
source venv/Scripts/activate

# PowerShell (if execution policy error)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
.\venv\Scripts\Activate.ps1
```

#### 5. Port Already in Use

```bash
# Run backend on different port
uvicorn app.main:app --reload --port 8001

# Run frontend on different port
npm run dev -- -p 3001
```

#### 6. Migration Issues

```bash
# Generate new migration
alembic revision --autogenerate -m "description"

# Apply all migrations
alembic upgrade head

# Downgrade one step
alembic downgrade -1
```
